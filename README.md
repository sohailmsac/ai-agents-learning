# AI Agents Learning

## Setup

### Environment Configuration

1. Create a `.env` file in the root directory of this project
2. Add your environment variables to the `.env` file:
   ```
   OPENAI_API_KEY=your_api_key_here
   ```

### Running the Examples

Navigate to the desired session folder and run the Python script:
```bash
python sessions/session-1/ex1_1_basic_chat.py
```

Make sure the `.env` file is present in the root before running any scripts.
